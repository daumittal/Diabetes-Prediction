# Diabetes-Prediction

In this project, we'll utilize the Python Keras library to build neural networks for predicting diabetes using patient health data. Our process involves loading and analyzing data with pandas, visualizing it through matplotlib and seaborn, and conducting data cleaning, preprocessing, and feature selection. Subsequently, we'll split the data into training, validation, and testing sets using NumPy. Once prepared, we'll train straightforward deep learning models using Keras, plot training curves, and evaluate models using scikit-learn.
